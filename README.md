## Descripción

ServiCerca es una aplicación pensada para conectar a usuarios con proveedores de servicios locales verificados. Permite buscar ofertas por categoría y zona, ver rangos de precio estimados, solicitar servicios directamente desde la publicación y confiar en la moderación que verifica la legitimidad de las ofertas.


## Características principales

- Búsqueda por categoría, ubicación y filtros (precio, valoración, nivel del proveedor).
- Publicación de ofertas por proveedores con campos estructurados.
- Solicitud de servicio desde la publicación (chat/solicitud integrada o contacto directo).
- Moderación manual de ofertas para evitar fraudes y contenido engañoso.
- Perfiles de usuario con niveles (Principiante, Profesional, Experto, Maestro).
- Visualización de la zona de cobertura (mapa simple o lista de barrios).
- Rango de precio estimado por oferta.



## Categorías

- Hogar: Plomería, Electricidad, Limpieza, Reparaciones generales
- Educación: Clases particulares, Tutorías, Preparación de exámenes
- Mascotas: Paseo, Cuidado, Veterinaria
- Tecnología: Reparación de equipos, Soporte técnico, Instalaciones
- Transporte: Mudanzas, Mensajería, Fletes

## Flujo de usuarios

1. Usuario abre la app y busca por categoría o ubicación.
2. Visualiza una lista de ofertas con precio estimado y zona.
3. Abre una oferta para ver detalles y puede solicitar el servicio.
4. Luego del servicio, el usuario puede dejar valoración y reseña.
